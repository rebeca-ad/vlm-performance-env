# VLM Performance Environment
This repository contains the benchmarking environment used for evaluating Vision-Language Models (VLM) for autonomous driving context analysis.

## Hardware Environment
- **Device:** Acer Nitro 5
- **RAM:** 32GB
- **Optimization:** 4-bit Quantization (BitsAndBytes)

## Performance Metrics
- **Moondream2 Decoding:** 35.39 ms/token
- **Qwen2.5-VL Decoding:** 86.03 ms/token
- **Image Loading Latency:** 10.52 ms
